# Termux Android 编译包 - 快速安装指南

## 在 Termux 中运行

打开 Termux，复制粘贴以下命令：

```bash
pkg update -y && pkg install -y git wget curl && \
git clone https://github.com/your-username/termux-android-builder.git ~/termux-android-builder && \
cd ~/termux-android-builder && \
bash install.sh
```

## 安装完成后

```bash
# 检查环境
bash env-check.sh

# 编译内核
bash build-kernel.sh -k ~/kernel -c vendor/defconfig
```
